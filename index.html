Step-by-Step Instructions

Follow these detailed steps to become a validator. Each step includes extra information so that even beginners can complete the process without any hassle.

I. Installing Solana CLI

For MacOS & Linux:

  1. Open your Terminal: - On MacOS, find Terminal in Applications > Utilities. - On Linux, open your preferred terminal application.
  2. Ensure prerequisites: - Make sure curl is installed on your system.
  3. Run the installation command: - Copy and paste the following command in your terminal: sh -c "$(curl -sSfL https://release.anza.xyz/v2.1.5/install)"
  4. PATH Variable: - If prompted at the end, update your PATH environment variable as instructed.
  5. Verify Installation: - Run solana --version to ensure the CLI is correctly installed.

For Windows:

  1. Run as Administrator: - Search for "Command Prompt" in the Start menu, right-click, and select "Run as Administrator".
  2. Download Installer: - Execute: cmd /c "curl https://release.anza.xyz/v2.1.5/agave-install-init-x86_64-pc-windows-msvc.exe --output C:\agave-install-tmp\agave-install-init.exe --create-dirs"
  3. Install CLI: - Run: C:\agave-install-tmp\agave-install-init.exe v2.1.5 - Follow any on-screen prompts and wait until installation finishes.
  4. Restart Command Prompt: - Close the admin prompt and open a new, regular Command Prompt.
  5. Verify Installation: - Run solana --version to confirm proper installation.

II. Creating Validator Keys

  1. Generate a Testnet Key: - On Windows, open Command Prompt and run: solana-keygen new -o %USERPROFILE%/testnet-validator-keypair.json - On Linux, run: solana-keygen new -o ~/testnet-validator-keypair.json - Save the seed phrase and key information securely.
  2. Generate a Mainnet Key: - On Windows, run: solana-keygen new -o %USERPROFILE%/mainnet-validator-keypair.json - On Linux, run: solana-keygen new -o ~/mainnet-validator-keypair.json - Again, securely store the seed phrase and key details.

III. Signing Keys for the Delegation Program

  1. Visit the Delegation Program Website: - Open https://solana.org/delegation-program in your browser.
  2. Sign Your Keys: - Use the command below (replace PATH_TO_KEYPAIR with your key‚Äôs file path):
    solana sign-offchain-message BNIYTO --keypair PATH_TO_KEYPAIR
  3. Enter Signed Keys: - Copy the signed output and paste it into the corresponding fields on the website.

IV. KYC (Know Your Customer) Verification

  1. Fill Out the Form: - Enter your real personal details in English. Include your Discord username and your country of residence.
  2. Email Verification: - Check your email (and spam folder) for a verification code, then enter it into the form.
  3. Complete the Agreement: - Fill in your full name, city, and other required information in the agreement form.
  4. Upload Identification: - Scan or take a clear photo of your passport or driver‚Äôs license. If the image quality is poor, you may need to re-upload.

V. Purchasing a Server for Your Validator

  1. Choose a Payment Method: - Options include overseas bank cards via bitfree.us or cryptocurrency (e.g., via MetaMask). Ensure you check network details carefully.
  2. Select a Server: - Your server should meet the following requirements:
    • Processor: AMD Ryzen 9 5950X (or higher)
    • RAM: Minimum 64GB (preferably 128GB)
    • Storage: 2 x 1TB NVMe SSD in RAID 0 configuration
    • OS: Ubuntu 20.04 or 22.04 (mandatory)
    • SSH Key: Use the SSH public key you generated earlier
  3. Register and Verify: - Complete the registration process on your chosen provider‚Äôs site, verify your account via email, and fill in your profile with accurate details.
  4. Purchase and Wait: - After paying, wait for 3-7 days to receive your server credentials.
